# James Hoang - Developer Portfolio

Live site: <https://jameshoang.com>

## Overview

This repository contains the source code for my personal developer portfolio. The site showcases my projects, technical skills, and experience, with a focus on backend systems, automation, and AI-assisted applications.

The portfolio is designed to be simple, fast, and recruiter-friendly, serving as a central hub that links my projects, GitHub repositories, and professional profiles.

## Features

- Project-focused layout highlighting real-world applications and hackathon work
- Clear separation between projects, skills, and contact information
- Deployed and maintained as a production website
- Responsive design for desktop and mobile viewing

## Tech Stack

- Frontend: React, JavaScript / TypeScript
- Styling: CSS
- Deployment: Vercel
- Version Control: Git and GitHub

## Highlighted Projects

- Quizzz - Interactive, AI-assisted quiz generation and play platform
- Trip Planner - AI-powered travel planning app built in a 24-hour hackathon
- Escape from Windows - Interactive puzzle adventure game simulating a Windows XP environment
- Novel to Audiobook - Automated pipeline for converting web novels into audiobooks

Each project links to its live demo and/or GitHub repository with detailed documentation.

## Getting Started (Local Development)

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
```
